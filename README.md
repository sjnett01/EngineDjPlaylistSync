# Engine DJ Playlist Sync 

A tool to "Syncronize"  Music folders in Engine DJ
- Can Add files added to folders directly without recreating playlists
- Has a built in tool to analyze tracks in the application (should be quicker than the official tool, but is not as accurate, but close enough to be useful
- Has the ability to scan for missing tracks, and update the database with the new path, or delete the missing files if they have been deleted intentionally
